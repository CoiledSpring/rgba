# GBA Core (Gameboy Advance)

The gba core is the second core to have been written. It's currently in early stages.

## Supported features

 - Nothing at this point
 
## Planned features

 - ARM CPU emulation
 - Debugger (disassembly, breakpoints, watchpoints)
 - Misc system emulation
 - Graphics emulation
 - Sound emulation

# DMG Core (Gameboy)

The dmg core is the first core to have been written. It's still being written.

## Supported features

 - Gameboy CPU
 - DMG video
 - Timer, DMA, Input
 - No MBC, MBC1 and MBC3 (without RTC)
 - Basic debugger (stepping, watchpoints)
 
## Planned features

 - Sound
 - GBC support
 - RTC support for MBC3
 - Watchpoints
